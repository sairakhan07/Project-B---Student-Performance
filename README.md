The aim of this project is to design and implement a console-based Java-based object-oriented application to manage student marks and grades for university modules on the Bolton University BSc(Hons) Computing course (3 year programme). The system should be user-friendly, efficient, and reliable. 
The Object Oriented design should be robust for representing students, modules, marks and grades. It should utilise appropriate object oriented principles (encapsulation, inheritance, polymorphism), and implement data structures (e.g. arrays, lists, maps) to store student information, module details and marks. Consider data validation and error handling to ensure data integrity. 
The console-based user interface will allow users to: 
● Add, edit and delete students and modules; 
● Input and update student marks for each module; 
● Calculate and display student grades based on predefined grading criteria; 
● Calculate the mean mark, minimum mark and maximum mark for each module ● Display a grade profile (the percentage of students obtaining each grade) per module 
Thoroughly test the application to ensure it functions correctly and handles edge cases. Debug any issues identified during testing. 
Adhere to coding standards and best practices. Write clean, well-commented and readable code. Utilise appropriate exception handling mechanisms.
